## Install

```bash
npm install -g memer-cli
```

After installation you can run `memer` from anywhere in your terminal.

----------

## Usage

### List available memes

```bash
memer list
```

Shows all built-in and downloaded memes.

----------

### Play a specific meme

```bash
memer fahh  
memer among-us  
memer bruh
```

Plays the meme directly in your terminal.

----------

### Play a random meme

```bash
memer random
```

Let fate decide.

----------

## How it Works

-   Cross-platform (Windows / macOS / Linux)
    
-   Plays audio without opening a media player
    
-   Caches optimized audio for instant replay
    
-   Works fully offline after installation

----------

$ memer list
